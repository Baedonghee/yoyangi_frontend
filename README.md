# yoyangi_frontend

`shoecatch-front-platform` 스타일을 기준으로 잡은 `pnpm` 워크스페이스 + `turbo` 모노레포 골격입니다.

## Structure

```text
apps/
  web/     # Next.js
  admin/   # Vite + React
packages/
  eslint-config/
  prettier-config/
  theme/
  typescript-config/
  ui/
  utils/
```

## Commands

```bash
pnpm install
pnpm dev
pnpm dev:web
pnpm dev:admin
```
